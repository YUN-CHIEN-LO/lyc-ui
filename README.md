## 專案規範

### Git Commit
採用 [AngularJS Git Commit Message Conventions](https://docs.google.com/document/d/1QrDFcIiPjSLDn3EL15IJygNPiHORgU1_OOAqWjiDU5Y/edit) 約定式提交

可使用的 type：

| Type | 功能 |
|:-:|:-:|
| `feat` | 新開發的項目 |
| `fix` | 錯誤修正 |
| `docs` | 更新文件 |
| `style` | 程式碼格式化 |
| `refactor` | 程式碼重構 |
| `perf` | 效能改善 |
| `test` | 更新測試案例 |
| `build` | 變更建構設置或外部依賴 |
| `ci` | 更新 CI 或 script 指令 |
| `chore` | 其他雜事 |
| `revert` | 程式碼退回 |
